Stories

How a Steady day can feel.

Not reviews — day sketches. The kind of mornings and evenings Steady is built for.

Self-focus · Monday

“Work until lunch. Then Focus.”

You set the schedule Sunday night. Monday morning the phone is already in Work — Slack and docs, not the feed. At noon it flips to Focus. Fun waits for a real unlock. By evening you’ve still got minutes left… or you don’t. Either way, the day had edges.

Parent · After school

“To-dos first. Then Ask.”

You sent three to-dos this morning. They claim each one. You Approve. Fun unlocks. They tap Ask for twenty more minutes — or bedtime still locks at nine. No silent bargaining with Settings.

Self-focus · Weekend

“I needed a Day off.”

Travel day. Maps, tickets, group chats. You unlock, take a Day off, live freely, then end it when you’re home. The plan comes back — without a guilt spiral.

Family · Night

“Sleep means Sleep.”

Schedule hits Sleep. The phone becomes quiet infrastructure — not a second bedroom light. Morning opens Essentials or Learning. The night stayed the night.

Getting started

How do I install Steady?

Download the APK on this site → open the file → allow install from this browser if Android asks → open Steady.

How do I pick self-focus vs parenting?

On first setup, choose for me if it’s your phone, or child / parent for family. You can switch roles later in Settings.

Do I need a computer?

No for everyday use. Protection turns on right on the phone. A computer is only for the optional uninstall-lock path.

Is Steady free?

Yes. Download here. Current builds don’t put a subscription wall in front of the door.

What data does Steady collect?

No ads and no analytics SDK. Most locks and timers stay on the phone. Family pairing, Approvals, and to-dos can sync when you use those features. Full plain-language detail: privacy policy.

Android says unknown / blocked — what now?

That’s normal when you install from a website (not Play Store). Allow installs from your browser or Files, then open the APK again. The warning is expected — see the note under Download.

Where do I start if I’m in a hurry?

Download → install → pick a role → finish protection → apply Focus or set a short schedule. Refine later.

For yourself (self-focus)

How do I use Steady just for me?

Setup → This phone is for me. Set a PIN (and optional friend Approve). Turn protection on. Build a day schedule for Work, Focus, Fun, and Sleep.

How do I set a day schedule?

Settings (unlock first) → Day schedule. Set hours for Work, Focus, Learning, Fun, and Sleep.

How do I make Steady my Home screen?

On Steady Home, choose Make Steady my Home screen and pick Steady as default Home.

How do I rearrange apps or the dock?

Long-press an app — rearrange, star into the dock, reorder or remove dock icons, or reset the layout.

What is Day off?

A sunny pause. Unlock with your codes, take open phone time, then end Day off when you’re ready.

How do I ask a friend to Approve overtime?

Create a family code in Settings and give it to a friend on a second phone. When you want Fun past the gate, you Ask — they Approve.

What if I just want less scrolling tonight?

Apply Focus · anti-addict or start Sleep early on your schedule.

Can Steady block itself by accident?

No. Steady stays reachable so you can Ask, unlock, and open Settings.

For parents

How do I set up a child phone?

Install Steady → choose child → set unlock checks → run the protection wizard → pair with a parent phone using a family code. Full walkthrough: setup guide.

How does Entertainment approval work?

Fun isn’t free by default. The child taps Ask. You Approve or Deny from the parent phone inbox.

How do I set daily minutes / bedtime?

On the parent phone, unlock Settings. Set allowances, bedtime, and forced breaks. Send rules to the kid phone when you change them.

What is the family to-do list?

Parent creates to-dos and sends them to the kid phone. Kid taps I did it. Parent Approves or sends back. When the list is approved, Fun reward minutes unlock. Optional: require to-dos before Fun Ask.

How do chores and homework fit in?

Chores can earn entertainment minutes when marked done. Optional homework confirm asks before Fun unlocks. Family to-dos are the structured list version of the same idea.

What are school hours?

In parent settings, weekday school hours can force Focus and block Fun — like a built-in school-time mode.

How do kids update Steady?

When an update is ready, Home can show an update icon — so they can update without digging locked Settings.

Can we use one phone for kid + parent?

Yes — both mode. Two phones is stronger. Keep the PIN secret.

What if my kid turns off Accessibility?

Steady notices. It locks and asks for Admin / Accessibility back. Optional uninstall lock is available for the serious path.

Can I manage kids from a computer?

Yes — open the Family dashboard. Sign in with Google or an email one-time code (SMS is not free, so Steady does not text). Same Google on the kid phone links the family. Sync still uses GitHub only.

Can a kid turn Steady off in Settings?

Without parent unlock, Steady blocks Accessibility, Admin, Users, Home picker, and About (Developer Options). Tamper repair only opens Accessibility + Device Admin — not the rest of Settings. For uninstall / Safe Mode / factory reset, set the kid phone as Device Owner.

Steady feels slow or says isn’t responding?

Update to the latest build. 1.7.7+ lightens Settings scans and background sync so the Home screen stays responsive.

Do browsers work in Work or Focus?

Browsers need Fun — not Work or Focus. That keeps “just checking something” from becoming a scroll hole.

Day to day

How do the five profiles work?

Focus = short anti-addict window. Essentials = calls/texts/maps/camera. Work / Learning = only those apps. Entertainment = Fun with a gate. Schedules can also enter Sleep.

Does every minute burn my timer?

No. Steady counts screen-on, visible apps. Screen-off calls and background music don’t burn minutes.

How do I change language?

Settings → language. English, Hebrew, Yiddish, Spanish, Russian — RTL for Hebrew and Yiddish.

How do I update Steady?

Settings → check for updates, or use the Home update icon. Files can save under Downloads → Steady.

What unlock codes can I use?

PIN, draw-a-pattern lock, passphrase, security question, Steady Face, parent stamp, remote Approve, or a friend’s phone unlock. Pick 1–3 steps.

Does Call open the normal Phone app?

Yes. Call opens the stock Phone app.

What’s on the dock?

Phone, Messages, Camera, Gallery by default — and you can star other apps into the dock in edit mode.

Can widgets freely switch profiles?

No. Profile switches stay intentional — so a widget tap can’t quietly undo your day plan.

Little snags

Install fails over an old Steady?

Uninstall the old build once, then install the newest APK. Android is picky about signing keys.

What is Quick Focus?

A timed Focus sprint (15 / 25 / 45 minutes) from Home. Fun stays locked until it ends — the same idea as Deep Focus in apps like Opal.

What is the mindful pause?

Before Ask for Fun, Steady waits a few breaths so you can cancel autopilot taps. Turn it off under Appearance if you want.

How do I turn Steady off completely?

Take a Day off first (unlock with your codes). Then deactivate protection in system settings if you want to uninstall.

Protection keeps asking to turn back on?

That’s on purpose. Turning Admin or Accessibility off is the classic escape hatch — Steady fights that so the day stays steady.

Update downloaded but install failed?

Look in Downloads → Steady for the file and try again. The Home update icon also helps when a new build is ready.
