What Steady is now

Not three modes.
A full day system.

Steady has grown past a simple Work / Learning / Fun switch. Today it’s child, parent, or both-on-one-phone roles; scarce daily minutes; parent-gated entertainment; bedtime locks; chores that earn time; and approvals over Nearby Bluetooth or the cloud.

01

Scarce phone time

Daily allowances per profile. The timer only burns for screen-on, visible apps — screen-off calls and background music don’t eat minutes.

02

Entertainment with a gate

Kids ask. Parents approve or deny — from the parent phone inbox, with optional homework confirm and multi-step parent checks.

03

Strong on the kid phone

Device Admin + Accessibility + usage access work without a PC. Full Device Owner (uninstall lock) is optional advanced.

Under the hood

What parents actually get today.

Child · Parent · Both

Two phones is strongest. Same-phone “both” mode exists for Approve + locks on one device — with a clear “keep the PIN secret” warning.

Family code pairing

Parent creates a 6-digit family code. Child enters it. Parent can send rules; child can refresh. Approvals also via Nearby Bluetooth (no Wi‑Fi needed).

Parent checks (1–3 steps)

PIN, Samsung-style pattern, passphrase, security question, Steady Face (in-app camera), parent stamp, remote approve, or parent-phone biometrics for remote asks.

Bedtime & forced breaks

Bedtime window locks the phone. Continuous-use cooldowns force a rest. Parents set hours and lengths in Settings.

Chores & homework gate

Chores can earn entertainment minutes. Optional “did they finish homework?” confirm before unlock.

Family web filter

Local DNS VPN blocks listed adult and gambling domains, and known bypass VPN apps — not a full content proxy, but a real shield.

Enforcement that sticks

Foreground service + Accessibility close blocked apps. With Device Owner: gray-out icons, block uninstall, restrict sideload/VPN config.

Languages that fit home

English, Hebrew, Yiddish, Spanish, Russian — RTL for Hebrew and Yiddish. Text size and per-app language in Settings.

In-app updates

Check for updates and install from Settings. Crash reports can sync privately for fixes.

Getting started

From download to “You are ready.”

  1. Install the APK Sideload on child and parent phones. No Play Store required.
  2. Pick the phone’s role Child, Parent, or Both. Child phones run the protection wizard on-device.
  3. Protect without a PC Device Admin → Accessibility → Usage access → Family web filter → battery unrestricted. Device Owner via ADB is optional.
  4. Pair & live in profiles Create a family code, connect, set daily minutes / bedtime, then run Focus, Essentials, Work, Learning, or Entertainment with approval.

FAQ

Straight answers.

Is Steady free?

Yes. Download the APK here. No subscription wall on current builds.

Do I need a computer?

No for normal use. Device Admin + Accessibility + usage access are on-phone. A computer is only for optional Device Owner (uninstall lock) via ADB.

What’s new vs the old “three modes” site?

Steady now ships Focus and Essentials profiles, child/parent/both roles, bedtime and breaks, chores, homework gate, family codes, Nearby approvals, Steady Face / pattern checks, and a local DNS web filter.

Can kids bypass it?

Strong path: Device Admin + Accessibility + PIN/parent checks. Device Owner adds uninstall lock. No tool replaces parenting — Steady supports your rules.

Two phones or one?

Two phones is best (quiet parent + locked child). Both-on-one-phone works for Approve + locks together, but keep the PIN secret — it’s weaker.

Which languages?

English, Hebrew, Yiddish, Spanish, Russian — RTL for Hebrew and Yiddish.
