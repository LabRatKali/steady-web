Getting started

How do I install Steady?

Download the APK on this site → open the file → allow install from this browser if Android asks → open Steady. No Play Store trip needed. That’s it — you’re in.

How do I pick self-focus vs parenting?

On first setup, Steady asks who this phone is for. Choose for me if it’s your phone and you want your time back. Choose child / parent for family setups. You can switch roles later in Settings if you need to.

Do I need a computer?

Nope for everyday use. Turn on Device Admin, Accessibility, usage access, and the web filter right on the phone. A computer is only for the optional Device Owner “uninstall lock” path.

Is Steady free?

Yes. Download here. Current builds don’t put a subscription wall in front of the door.

Android says the app is unknown / blocked — what now?

That’s normal when you install from a website. Allow installs from your browser or Files, then open the APK again. Steady isn’t from Play — it’s from this page, on purpose.

For yourself (self-focus)

How do I use Steady just for me?

Setup → This phone is for me (focus — get my time back). Set a PIN (and optional friend Approve). Turn protection on. Then build a day schedule so Work, Focus, Fun, and Sleep happen when you meant them to.

How do I set a day schedule?

Open Settings (unlock with your codes) → Day schedule. Set hours for Work, Focus, Learning, Fun, and Sleep. Steady flips profiles from the clock — including a full Sleep lock overnight. Tiny automation, big calm.

How do I make Steady my Home screen?

On the Steady home screen, tap Make Steady my Home screen and pick Steady as the default Home app. Home button comes back here; allowed apps show as a grid. Steady becomes where you live — not a nag you dodge.

How do I ask a friend to Approve overtime?

Create a family code in Settings and give it to a friend on a second phone (parent / accountability friend role). When you want Fun past the gate, you Ask — they Approve. Kind of like a gym buddy, but for your attention.

How do I pause Steady so I’m not stuck while testing?

Apply Essentials (or open Settings) → unlock → tap Pause Steady now. Locks stop. Resume when you’re ready — or deactivate Device Admin and uninstall if you’re done testing.

What if I just want less scrolling tonight?

Apply Focus · anti-addict or start Sleep early on your schedule. Short clock. Clear stop. Tomorrow you can loosen — tonight you get your evening back.

For parents

How do I set up a child phone?

Install Steady → choose child → set 1–3 parent checks → run the protection wizard (Admin, Accessibility, usage, web filter). Pair with a parent phone using a family code. Full walkthrough lives on the setup guide.

How does Entertainment approval work?

Fun isn’t free by default. The child (or self-focus user) taps Ask. You see it on the parent phone inbox — Approve or Deny. Approvals can also arrive nearby, phone-to-phone.

How do I set daily minutes / bedtime?

On the parent phone (or combined), open Settings after unlocking. Set allowances per profile, bedtime hours, and forced breaks. Send rules to the kid phone when you change them.

How do chores and homework fit in?

Chores can earn entertainment minutes when marked done. Optional homework confirm asks “did they finish?” before Fun unlocks. Small gates, better habits.

Can we use one phone for kid + parent?

Yes — both mode. It works, but it’s weaker than two phones. Keep the PIN secret. Two phones is the cozy, strong setup.

What if my kid turns off Accessibility?

Steady notices. It locks and asks for Admin / Accessibility back — because turning those off is the classic escape hatch. Optional Device Owner adds uninstall lock for the serious path.

Day to day

How do the five profiles work?

Focus = short anti-addict window. Essentials = calls/texts/maps/camera. Work / Learning = only those apps. Entertainment = Fun with a gate. Schedules can also enter Sleep.

Does every minute on the phone burn my timer?

No — and that’s the nice part. Steady counts screen-on, visible apps. Screen-off calls and background music don’t nibble your day away.

How do I change language?

Settings → language. English, Hebrew, Yiddish, Spanish, Russian — with RTL for Hebrew and Yiddish. Text size lives there too.

How do I update Steady?

In Settings, tap check for updates / download & install. You’ll see a clear “What’s in this update” note. Or grab the newest APK from this website anytime.

Can Steady block itself by accident?

No. Steady stays reachable so you can Ask, unlock, and open Settings. The app that keeps you honest shouldn’t lock you out of itself.

What unlock codes can I use?

PIN, draw-a-pattern lock, passphrase, security question, Steady Face, parent stamp, remote Approve, or a friend’s phone unlock. Pick 1–3 steps — as gentle or as firm as you need.

Little snags

Install fails over an old Steady — help?

If an older build used a different signing key, uninstall the old Steady once, then install the latest APK. Android is picky about that; Steady isn’t mad at you.

How do I turn Steady off completely?

Pause Steady first (Essentials → Settings → Pause). Then open Device Admin and deactivate Steady if needed, then uninstall. Don’t fight the locks — pause, then exit kindly.
