Getting started

From download to “you are ready.”

  1. Install on both phones Same APK for child and parent. No Play Store required.
  2. Set up the child device Child mode, parent PIN, protection level, then Device Owner via one adb command.
  3. Pair the parent Create a pair code, connect, and start approving entertainment when asked.
  4. Live in modes Work → Learning → Entertainment with approval. Habits stick because the phone cooperates.

FAQ

Straight answers.

Is Steady free?

Yes. Download the APK from this site. There’s no subscription wall on the current builds.

Do I need Google Play?

No. Steady is sideloaded. Allow install from your browser once, then open the APK.

Why USB and a computer?

Full Device Owner protection needs one adb command. The app shows the exact line. Day-to-day use is on the phones after that.

Can kids bypass it?

Device Owner + parent PIN + usage access is the strong path. No tool replaces parenting — Steady supports your rules.

What about privacy?

Steady centers on local control and parent pairing. Optional sync supports updates and parent approvals. No ads in the app.

Which languages?

English, Hebrew, Yiddish, Spanish, and Russian — RTL for Hebrew and Yiddish.
