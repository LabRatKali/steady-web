Pending asks

Your child asked for Fun, an app, a website, or a profile change. For time asks, pick minutes. For profile / settings asks, tap Allow or Don’t allow. Dismiss clears the ask forever.

Install asks

New apps the kid tried to install — Approve lets the app run, Deny keeps it blocked. Needs Install approval turned on under Controls.

Routines

One-tap profiles inspired by Qustodio schedules and Opal focus sessions. Each pushes to the kid phone right away.

Instant actions

Family Link / Qustodio style — lock Fun, pause the web filter, or clear a forced mode. Each action pushes immediately.

Modes & guards

Each switch is one rule for the kid phone. Example: School hours locks Fun during school. Install approval means new apps need your OK. Flip what you want, then tap Save & push at the bottom.

Force mode / profiles

Push a mode onto the kid phone right away. Focus = calm tools only (no daily clock). Work / Learn / Fun use the minutes you set under Budgets. Clear stops forcing a mode.

Pause & unlocks

Day off turns Steady soft for a while so the kid can use the phone freely. Settings unlock lets them open Android Settings for a few minutes (dangerous pages still go Back). Break Steady pauses locks briefly. Always tap Save & push after.

Daily budgets & bedtime

How many minutes of Work, Learn, and Fun each day. Focus has no daily clock. Bedtime locks Fun overnight. School hours use the start/end times. Edit, then Save & push.

Kid apps

Always = never blocked. Focus / Work / Learn / Fun = only in that mode. Never = blocked. Tapping a bucket saves that app right away (no Save & push needed).

Family to-dos

Add chores the kid should finish. Mark done sends Approve to the kid phone right away so Fun can unlock when you require to-dos.

Ideas for parents

Small habits that keep Steady simple for everyone.

  • Name each kid phone so you never hunt for a device ID.
  • Use Day off for travel days — apps stay colorful and open.
  • Approve asks from this Pending asks tab; the phone picks them up in about 15 seconds or when the kid taps Refresh.
  • Focus on the kid phone opens almost everything; switching Focus still needs your PIN or an ask here.
