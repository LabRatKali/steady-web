Steady

Parent setup guide

From install to “you are ready” — child phone, parent phone, Device Owner.

Child phone

  1. Install Steady on the child phone.
  2. Open Steady and choose This phone is for a child.
  3. Pick a parent PIN (4+ digits) and a security question.
  4. Choose 1-step, 2-step, or 3-step protection.
  5. Connect the phone to a computer with USB debugging.
  6. Copy the command from the app and run it in a terminal: 
    adb shell dpm set-device-owner app.steady.android/app.steady.deviceowner.SteadyDeviceAdminReceiver
  7. On the phone, allow Usage access for Steady.
  8. Tap You are ready.

Parent phone

  1. Install the same Steady APK.
  2. Choose This phone is for a parent.
  3. In Settings, create a pair code and enter it on the child phone.
  4. Approve Entertainment requests from the parent home screen.

Modes

  • Work — work apps only
  • Learning — study apps only
  • Entertainment — needs parent approval first

Calls with the screen off and background music do not use up the timer.
