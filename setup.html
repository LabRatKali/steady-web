Setup guide

Short steps. Self-focus or parenting.

1. Install

  1. Download the APK.
  2. Allow install if Android asks.
  3. Open Steady.

2. Who is this phone for?

  • For me — self-focus, schedules, optional friend Approve
  • Child — locks and Ask for Fun
  • Parent / friend — Approve inbox + family code
  • Both — one phone does both (weaker; keep the PIN secret)

3. Self-focus

  1. Choose for me.
  2. Set a PIN (optional friend Approve).
  3. Turn on protection on the phone (Admin, Accessibility, usage, web filter).
  4. Settings → Day schedule — Work / Focus / Learning / Fun / Sleep.
  5. Optional: Make Steady my Home screen.
  6. Need a break? Unlock → Day off. End it when ready.

4. Child phone

  1. Set unlock checks.
  2. Follow protection steps on the phone.
  3. Pair with a parent phone using a family code.
  4. Optional advanced uninstall lock (computer):
adb shell dpm set-device-owner app.steady.android/app.steady.deviceowner.SteadyDeviceAdminReceiver

5. Parent / friend phone

  1. Choose parent / accountability friend.
  2. Create a family code; enter it on the other phone.
  3. Approve Fun from the inbox.

6. Profiles

  • Focus · anti-addict
  • Essentials
  • Work · Learning · Entertainment
  • Sleep from schedules

Long-press apps on Steady Home to rearrange or edit the dock.
