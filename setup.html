Steady

Setup guide

Short steps for self-focus or parenting. Day schedules, Steady as Home, Day off, and protection on the phone.

1. Install

  1. Download the APK from the Steady site.
  2. Allow install from this browser if Android asks.
  3. Open Steady.

2. Who is this phone for?

  • For me — self-focus, day schedules, optional friend Approve
  • Child — locks and Ask for Entertainment
  • Parent / friend — Approve inbox + family code
  • Both — one phone does both (weaker; keep the PIN secret)

3. Self-focus

  1. Choose This phone is for me.
  2. Set a PIN (optional friend Approve on a second phone).
  3. Turn on protection: Device Admin → Accessibility → Usage access → web filter → battery.
  4. Settings → Day schedule — Work / Focus / Learning / Fun / Sleep.
  5. Optional: Make Steady my Home screen. Long-press apps to rearrange or edit the dock.
  6. Need open phone time? Unlock → Day off. End it when you’re ready.

4. Child phone

  1. Choose child and set unlock checks (1–3 steps).
  2. Follow the protection wizard on the phone — no computer needed for normal use.
  3. Pair with a parent phone using a family code.
  4. Optional advanced uninstall lock (computer only):
adb shell dpm set-device-owner app.steady.android/app.steady.deviceowner.SteadyDeviceAdminReceiver

5. Parent / friend phone

  1. Choose parent / accountability friend.
  2. Create a family code; enter it on the other phone.
  3. Approve or Deny Fun from the inbox.
  4. Set allowances, bedtime, and breaks — then send rules to the kid phone.

6. Profiles

  • Focus · anti-addict — short daily clock for less mindless use
  • Essentials — calls, texts, maps, camera
  • Work / Learning — only those apps
  • Entertainment — Fun with unlock / Approve
  • Sleep — from schedules, overnight rest lock

7. Updates

  • Settings → check for updates
  • Or use the Home update icon when a new build is ready
  • Files can land under Downloads → Steady
