Steady

Parent setup guide

Matches the app as it ships now — roles, on-phone protection, pairing, profiles.

1. Install

  1. Download the Steady APK on the child phone and the parent phone.
  2. Allow install from your browser / Files if Android asks.
  3. Open Steady.

2. Choose this phone’s role

  • Child — locks, profiles, timer, asks for entertainment
  • Parent — quiet remote: Approve inbox, family code, send rules
  • Both — kid home + Parent Approve on one phone (weaker; keep the PIN secret)

3. Child phone — protect (no computer needed)

  1. Set up 1–3 parent checks (PIN, pattern, passphrase, question, Steady Face, etc.).
  2. Follow the protection wizard on the child phone:
    • Allow restricted settings (Android 13+ if asked)
    • Turn on Device Admin
    • Turn on Steady Accessibility
    • Allow Usage access
    • Turn on Family web filter (DNS VPN)
    • Battery unrestricted for Steady
  3. Optional advanced: Device Owner via computer (ADB) if you need uninstall lock.
  4. Tap Start with 5:00 min entertainment (recommended) for a short demo day, or You are ready.

ADB Device Owner command (optional):

adb shell dpm set-device-owner app.steady.android/app.steady.deviceowner.SteadyDeviceAdminReceiver

4. Parent phone

  1. Choose This phone is for a parent.
  2. In Settings, Create / refresh family code.
  3. On the child phone, enter the 6-digit family code.
  4. Parent: Send rules to kid. Child: Refresh rules from parent when needed.
  5. Approve Entertainment requests from the parent home inbox (or Nearby Bluetooth).

5. Profiles & parent settings

  • Focus · anti-addict — work + learning, short daily clock
  • Essentials only — calls / texts / maps / camera
  • Work — work apps only
  • Learning — study apps only
  • Entertainment — needs parent unlock

In parent Settings you can set daily minutes, bedtime, forced breaks, chores (+10 min), homework confirm, languages, and check for updates.

Calls with the screen off and background music do not use up the timer.
