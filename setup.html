Setup guide

For yourself (self-focus) or for parenting — matches Steady as it ships now.

1. Install

  1. Download the Steady APK.
  2. Allow install from your browser / Files if Android asks.
  3. Open Steady.

2. Choose who this phone is for

  • This phone is for me (focus — get my time back) — adult self-discipline, schedules, optional accountability friend
  • This phone is for a child — locks, profiles, asks for Fun
  • This phone is for a parent / accountability friend — quiet Approve inbox + family code
  • This phone is for both — kid + Approve on one phone (weaker; keep the PIN secret)

3. Self-focus path

  1. Pick for me.
  2. Set unlock checks (PIN recommended; optional friend Approve).
  3. Turn on protection: Device Admin + Accessibility + usage access + web filter. Leave Admin and Accessibility on.
  4. In Settings → Day schedule, set Work / Focus / Learning / Fun / Sleep hours.
  5. Optional: create a family code and give it to a friend on a second phone for overtime Approves.
  6. Optional: tap Make Steady my Home screen and choose Steady as default Home.

4. Child phone path

  1. Set up 1–3 unlock checks (PIN, pattern lock, Steady Face, and more).
  2. Protection wizard (no computer needed):
    • Restricted settings (Android 13+ if asked)
    • Device Admin
    • Steady Accessibility
    • Usage access
    • Family web filter
    • Battery unrestricted
  3. Optional advanced: full uninstall lock with a computer (only if you need it).
  4. Finish with demo budgets or You are ready.

Optional computer command for that advanced lock:

adb shell dpm set-device-owner app.steady.android/app.steady.deviceowner.SteadyDeviceAdminReceiver

5. Parent / friend phone

  1. Choose parent / accountability friend.
  2. Create a family code; enter it on the child (or self-focus) phone.
  3. Send rules when needed; Approve Fun requests from the inbox (or nearby on another phone).

6. Profiles & Pause

  • Focus · anti-addict — short daily focus window
  • Essentials only — calls / texts / maps / camera
  • Work / Learning / Entertainment
  • Sleep (from schedules) — full rest lock

Pause Steady (testing): Apply Essentials → unlock Settings → Pause Steady now. Locks stop so you can test or uninstall.

Calls with the screen off and background music do not use up the timer.
